# Анимация яблони - v.0.0.1

## Описание

Создана уникальная анимация для яблони, которая отличается от анимаций вишни и груши. Анимация реализует плавное покачивание верхней части дерева с сохранением неподвижности нижней части.

## Технические характеристики

### Отличия от других деревьев

**Вишня и груша** используют анимацию `natural-tree-sway`:
- Техника: `rotate` + `translateX` (для изгиба ствола)
- Амплитуда: ±1.5°
- Точка трансформации: `transform-origin: 50% 95%`
- Длительность: 6-6.5 секунд
- Timing function: `cubic-bezier(0.25, 0.1, 0.25, 1)`

**Яблоня** использует новую анимацию `apple-gentle-sway`:
- Техника: `scaleY` + `rotate` (мягкое растяжение + поворот)
- Амплитуда: ±2.5° для rotate, 1.02 для scaleY
- Точка трансформации: `transform-origin: 50% 90%`
- Длительность: 7 секунд
- Timing function: `ease-in-out`

### Ключевые особенности

1. **Мягкое растяжение**: scaleY создает эффект "дыхания" дерева
2. **Плавный поворот**: больше амплитуда чем у других деревьев
3. **Более длинная длительность**: 7 секунд против 6-6.5
4. **Другая точка вращения**: 90% против 95% (больше верхней части в движении)

## Реализация

### CSS анимация

```css
/* Новая анимация яблони - плавное покачивание с растяжением */
@keyframes apple-gentle-sway {
    0%, 100% { 
        transform: rotate(0deg) scaleY(1);
    }
    25% { 
        transform: rotate(2.5deg) scaleY(1.02);
    }
    50% {
        transform: rotate(1deg) scaleY(1.01);
    }
    75% { 
        transform: rotate(-2.5deg) scaleY(1.02);
    }
}

/* Применение анимации к яблоне */
.apple-tree .tree-crown {
    animation: apple-gentle-sway 7s ease-in-out infinite;
    transform-origin: 50% 90%; /* Точка вращения выше, чем у других деревьев */
}
```

### Применение к яблоне

```css
.apple-tree .tree-crown {
    background: none;
    background-image: url('/static/img/7d125ebedaa511f08d6bf61bfd1d9f6b_1-removebg-preview.png');
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
    width: 504px;
    height: 529px;
    border-radius: 0;
    box-shadow: none;
    /* Заменяем animation: none на новую анимацию */
    animation: apple-gentle-sway 7s ease-in-out infinite;
    transform-origin: 50% 90%; /* Точка вращения в верхней части дерева */
}
```

## Визуальный эффект

### Движение частей дерева

- **Нижняя часть** (основание): практически неподвижна
- **Средняя часть**: легкое покачивание с растяжением
- **Верхняя часть**: основное движение с максимальной амплитудой

### Сравнение с другими деревьями

| Характеристика | Яблоня | Вишня | Груша |
|----------------|--------|--------|--------|
| Техника анимации | scaleY + rotate | rotate + translateX | rotate + translateX |
| Амплитуда поворота | ±2.5° | ±1.5° | ±1.5° |
| Дополнительный эффект | scaleY (1.01-1.02) | translateX (±2px) | translateX (±2px) |
| Длительность | 7 секунд | 6 секунд | 6.5 секунд |
| Точка трансформации | 50% 90% | 50% 95% | 50% 95% |
| Timing function | ease-in-out | cubic-bezier | cubic-bezier |

## Преимущества

1. **Уникальность**: каждое дерево имеет свой характер анимации
2. **Реалистичность**: имитация естественного движения под ветром
3. **Визуальное разнообразие**: предотвращает монотонность
4. **Производительность**: CSS анимации не нагружают JavaScript

## Совместимость

- **Браузеры**: поддерживается всеми современными браузерами
- **Мобильные устройства**: адаптивная анимация
- **Производительность**: hardware-accelerated CSS transforms
